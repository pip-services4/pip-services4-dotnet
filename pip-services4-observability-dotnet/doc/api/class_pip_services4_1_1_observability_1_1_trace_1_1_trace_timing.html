Detailed Description

Timing object returned by ITracer.beginTrace to end timing of execution block and record the associated trace.

var timing = tracer.BeginTrace("mymethod.exec_time");
try
{
...
timing.EndTrace();
}
catch (Exception err)
{
timing.EndFailure(err);
}
