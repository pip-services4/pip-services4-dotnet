Standard log levels. Logs at debug and trace levels are usually captured only locally for troubleshooting and never sent to consolidated log services.

Enumerator
None 

Nothing to log

Fatal 

Log only fatal errors that cause processes to crash

Error 

Log all errors.

Warn 

Log errors and warnings

Info 

Log errors and important information messages

Debug 

Log everything except traces

Trace 

Log everything.
